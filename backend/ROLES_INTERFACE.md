# Выбор ролей в интерфейсе Yandex Cloud

## Роли для выбора в интерфейсе

### 1. Категория `serverless.containers` (с точкой!)

В выпадающем списке найдите категорию **`serverless.containers`** (обратите внимание - с точкой, не дефис!).

Выберите следующие роли:
- ✅ **`editor`** - для развертывания контейнеров
- ✅ **`invoker`** - для работы API (вызов контейнеров)

⚠️ **Не выбирайте** категорию `serverless-containers` (с дефисом) - это другая категория!

### 2. Категория `container-registry`

Найдите категорию **`container-registry`** в списке и выберите:
- ✅ **`images.pusher`** - для загрузки Docker образов

## Пошаговая инструкция

1. В интерфейсе выбора ролей используйте поиск "Поиск" для быстрого поиска
2. Найдите и разверните категорию **`serverless.containers`** (с точкой)
3. Отметьте галочками:
   - `editor`
   - `invoker`
4. Найдите и разверните категорию **`container-registry`**
5. Отметьте галочкой:
   - `images.pusher`
6. Нажмите "Сохранить" или "Применить"

## Визуальная подсказка

В интерфейсе вы увидите:
- **`serverless-containers`** (с дефисом) - НЕ выбирайте эту категорию
- **`serverless.containers`** (с точкой) - ✅ ВЫБЕРИТЕ эту категорию

Внутри `serverless.containers` выберите:
- `editor` ✅
- `invoker` ✅

## Минимальный набор

Если хотите только необходимые роли:
1. `serverless.containers` → `editor` ✅
2. `serverless.containers` → `invoker` ✅  
3. `container-registry` → `images.pusher` ✅
