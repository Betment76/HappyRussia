# Быстрое создание PostgreSQL

## Через веб-консоль

1. Перейдите: https://console.cloud.yandex.ru/folders
2. Найдите **"Managed Service for PostgreSQL"** или **"Базы данных"**
3. Нажмите **"Создать кластер"**

## Параметры создания

### Основные настройки:
- **Имя:** `happyrussia-db`
- **Окружение:** `Production` или `Testing`
- **Версия:** `15` (или последняя)

### Класс хоста:
- **Минимальный:** `s2.micro` (1 vCPU, 4 GB) - для тестов
- **Рекомендуемый:** `s2.small` (2 vCPU, 8 GB) - для продакшена

### Диск:
- **Размер:** `10 GB` (минимум)
- **Тип:** `SSD`

### База данных:
- **Имя БД:** `happyrussia`
- **Пользователь:** `happyrussia_user`
- **Пароль:** Сгенерируйте и сохраните!

### Сеть:
- Выберите существующую сеть
- **Публичный доступ:** ✅ Включить

### Безопасность:
- Добавьте правило: `0.0.0.0/0` (или конкретный IP контейнера)

## После создания

1. Дождитесь статуса **"Running"** (5-10 минут)
2. Откройте кластер → вкладка **"Подключение"**
3. Скопируйте **FQDN хоста** (например: `c-xxxxx.rw.mdb.yandexcloud.net`)
4. Запомните порт: `6432`

## Строка подключения

```
postgresql://happyrussia_user:ВАШ_ПАРОЛЬ@c-xxxxx.rw.mdb.yandexcloud.net:6432/happyrussia
```

## Настройка контейнера

1. Откройте контейнер `happyrussia-api`
2. Создайте новую ревизию
3. Добавьте переменную окружения:
   ```
   DATABASE_URL=postgresql://happyrussia_user:ВАШ_ПАРОЛЬ@c-xxxxx.rw.mdb.yandexcloud.net:6432/happyrussia
   ```
4. Сохраните

## Готово!

После создания новой ревизии контейнер автоматически подключится к PostgreSQL.
