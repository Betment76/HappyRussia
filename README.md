# Моё Настроение

Приложение для отслеживания настроения по регионам России. Регионы соревнуются за звание самого счастливого!

## Описание

Приложение позволяет пользователям ежедневно фиксировать настроение выбором одного из 5 смайликов с привязкой к региону России. Главный экран отображает все регионы РФ, отсортированные по среднему уровню "счастливости".

## Технологии

- Flutter 3.x
- Provider для управления состоянием
- Geolocator для определения местоположения
- Dio для HTTP запросов
- SharedPreferences для локального хранилища

## Установка

```bash
flutter pub get
flutter run
```
